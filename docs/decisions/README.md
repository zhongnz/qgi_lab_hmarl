# Architecture / Project Decisions

Use this folder for decision records (ADR-style).

## Naming

`ADR-XXXX_title.md`

Examples:
- `ADR-0001_module_first_layout.md`
- `ADR-0002_forecast_sharing_strategy.md`

## Suggested sections

1. Context
2. Decision
3. Consequences
4. Alternatives considered
5. Status (`proposed`, `accepted`, `deprecated`)
