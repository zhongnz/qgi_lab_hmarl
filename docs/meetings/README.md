# Meeting Minutes

Store one markdown file per meeting.

## Naming

`YYYY-MM-DD_topic.md`

Examples:
- `2026-02-20_weekly-sync.md`
- `2026-03-05_forecasting-review.md`

## Recommended flow

1. Copy `../templates/meeting_minutes_template.md`
2. Fill it in during/after the meeting
3. Add follow-up items with owners and due dates
4. Link any related report or ADR
